cada elemento puede llevar el atributo id. Se utiliza para identificar de foma unica ese elemento de otros elementos de la pagina.Su valor debe comenzar con una letra y un giom bajo (no un numero o cualquier otro caracter). Es importande que no hay dos elementos en la misma pagina tienen el mismo valor para sus atributos ID (de lo contrario, el valor ya no es unico)

dar a un elemento una identidad unica le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la pagina

Poor ejemplo, es posible qeu se desee asignar un parrafo dentro de la pagina (quiza un parrafo que contiene una cotizacion de extraccion) un estilo diferente de todos los demas parrafos. En el ejemplo, el parrafo con el atributo ID cuyo valor es pullqote se hace en mayusculs usando css.
