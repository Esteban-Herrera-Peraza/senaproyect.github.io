<audio> HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus paginas. al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src: este atributo especificara la ruta de acceso al archivo de audio.

controls: este atributo indica si el reproductor debe mostrar los controles. si no utiliza este atributo, no se mostraran controles de forma predeterminada. tambien puede especificar sus propios controles mediante javascript.

autoplay: la presencia de este atributo indica que el audio debe empezar a reporducirse automaticamente. (Se considera una mejor practica permitir que los visitantes elijan reproductor de audio. )

loop: este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que halla finalizado
