la etiqueta video: para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro del elemento <video> (esto debe reemplaza el atributo arc en la etiqueta <video> de apertura.) tambien puede utulizar varios elementos <source> par especificar que el video esta disponible en diferentes formatos. (debido a un error en el ipad, debe proporcionar el video mp4 como el primer formato. De lo contrario, puede que no inicie.)

src: este atributo especificara la ruta deacceso al video.

type: se debe utilizar este atributo para decirle al navegador que formato es el video. de lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda) .

codecs: el codec que se uso para codificar el video se suministra dentro del atributo type. tenga encuenta el uso de comillas simples, asi como comillas bobles en el atributo de tipo, cuando se suministra.
